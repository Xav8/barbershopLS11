Perspectivă și transformări 3D

Efectuarea transformărilor CSS în spațiul 3D este puțin mai complexă. Trebuie să începeți prin a configura spațiul 3D oferindu-i o perspectivă, apoi trebuie să configurați modul în care elementele dvs. 2D se vor comporta în acel spațiu.

Perspectivă

Primul element de stabilit este perspectiva. Perspectiva este cea care ne dă impresia 3D. Cu cât elementele sunt mai departe de privitor, cu atât sunt mai mici.
